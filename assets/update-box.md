# Update box template

A "What changed" block with visible date markers, against recency loss.

## Frame

```markdown
## What changed?

* MM/YYYY: [Concrete content change: new number, new source, changed
  recommendation.]
* MM/YYYY: [Change.]
```

## Usage rules

- Two to four points, newest first.
- Enter only substantial changes; pure rewording does not count and risks a
  timestamp-gaming classification. See `../references/recency.md` and
  `../references/anti-patterns.md`.
- Each entry corresponds to a real text change on the same page.
